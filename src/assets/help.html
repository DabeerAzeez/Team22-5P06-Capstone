Use the Tooltips

Hover over most items in the GUI to see a short description of what the control does and how to use it. If you're unsure about anything, tooltips are the best place to start.

An operations manual will be available on the project GitHub page. If you do not have access, ask whoever provided you with this GUI for a copy of the manual.

FAQ / Common Issues

  • Controls not responding? Make sure the system is powered on and properly connected.
  • Data not updating? Check your sensor connections and refresh the interface.
  • Pressure/flow values look wrong? Recalibrate sensors and verify your input parameters.
